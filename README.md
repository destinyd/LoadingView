## 如何引用此组件：
### 安装
```
git clone https://github.com/mindpin/LoadingView
cd LoadingView
mvn clean install
```

### maven引用
在maven项目，pom.xml添加以下依赖引用：

```
<dependency>
<groupId>com.mindpin.android.loadingview</groupId>
<artifactId>loadingview</artifactId>
<version>(last version)</version>
<type>apklib</type>
</dependency>
```
